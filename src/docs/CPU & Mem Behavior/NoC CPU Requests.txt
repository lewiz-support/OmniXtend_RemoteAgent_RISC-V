//  Captured from Ariane RISC-V Core

//  Messages are separated with a row of dashes, with each flit of each message
//    represented in binary. Above each binary row are characters indicating the 
//    purpose of the bits as indicated in the flit descriptions below.
//
//  flit1  (i) 63:50 - CHIPID
//         (X) 49:42 - XPOS
//         (Y) 41:34 - YPOS
//         (f) 33:30 - FBITS
//         (L) 29:22 - PAYLOAD LENGTH  (# of QWords)
//         (t) 21:14 - MESSAGE TYPE    
//         (T) 13:06 - MSHR/TAG
//         (r) 05:00 - RESERVED/OPTIONS1
//  
//  flit2  (A) 63:16 - ADDRESS         (byte addressing, truncated to QWords (0-7 = QW0, 8-15 = QW1, etc)
//         (o) 15:03 - OPTIONS2        (S = Data Size Field)
//         (L) 02:00 - LAST BYTE COUNT (the last QWord contains byte 0 through this #)
//  
//          Data Size Field
//          Value  | Size (Bytes)  = 2^(n-1)
//          -------+-------------
//          3’b000 | 0
//          3’b001 | 1
//          3’b010 | 2
//          3’b011 | 4
//          3’b100 | 8
//          3’b101 | 16
//          3’b110 | 32
//          3’b111 | 64
//  
//  flit3  (i) 63:50 - SRC CHIPID
//         (X) 49:42 - SRC XPOS
//         (Y) 41:34 - SRC YPOS
